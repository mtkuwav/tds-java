# TDs Java

Chaque TD est dans son package propre (à part pour les TD 1 et 2 qui ne le nécessitaient pas), il suffit donc d'exécuter les fonctions `main` respectives des classes `Test` ou `Main` de chaque TD.

Les sujets sont accessibles dans le dossier `documents/sujets`, et les questions de connaissances du TD 1 dans le dossier `documents`.
