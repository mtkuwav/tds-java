## Fonctionnalités supplémentaires choisies
- Enchaînements automatique des calculs sans appuyer  sur `=`
- Gestion du clavier
- Arrondi ou formatage avancé des résultats

## Utilisation du clavier

### Chiffres
| Touche | Action |
|--------|--------|
| `0` à `9` | Saisit le chiffre correspondant |

### Opérateurs mathématiques
| Touche | Action | Équivalent |
|--------|--------|------------|
| `+` | Addition | Bouton + |
| `-` | Soustraction | Bouton - |
| `*` | Multiplication | Bouton × |
| `x` ou `X` | Multiplication (alternative) | Bouton × |
| `/` | Division | Bouton ÷ |
| `%` | Pourcentage | Bouton % |

### Point décimal
| Touche | Action |
|--------|--------|
| `.` | Ajoute un point décimal |
| `,` | Ajoute un point décimal (alternative) |

### Validation et effacement
| Touche | Action | Équivalent |
|--------|--------|------------|
| `Enter` ⏎ | Évalue l'expression et affiche le résultat | Bouton = |
| `=` | Évalue l'expression (alternative) | Bouton = |
| `Backspace` ⌫ | Supprime le dernier chiffre | Bouton DEL |
| `Delete` ou `Suppr` | Réinitialise la calculatrice | Bouton C |
| `Escape` ou `Échap` | Réinitialise la calculatrice | Bouton C |
| `C` | Réinitialise la calculatrice | Bouton C |
